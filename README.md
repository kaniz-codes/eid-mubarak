# Journey of Eid | Eid Mubarak

An interactive Eid website built with **HTML, CSS and vanilla JavaScript**, designed as a moonlit digital **pop-up storybook**.

## Experience

The visitor selects **Eid al-Fitr** or **Eid al-Adha** once on the opening screen. The site then renders only that journey: its illustrated six-page storybook, visual atmosphere, checklist, greeting content and references.

## Features

- One-time Eid choice saved with `localStorage`
- Animated storybook cover and six page-turning pop-up scenes
- Eid al-Fitr lantern/moon scenes or Eid al-Adha desert/path scenes only after selection
- Keyboard arrow and mobile swipe storybook navigation
- Takbir and Qur'anic dua cards with copy actions
- Interactive six-segment Dua Wheel, saved duas and private reflections
- Eid-specific checklist with saved progress
- Personalized English/Bangla greeting generator and PNG download
- English/Bangla interface toggle and dark/light theme toggle
- Floating **Made by Kaniz Fatema** signature and storybook-style author footer
- LinkedIn and GitHub follow buttons
- Accessible controls, modal, accordion and reduced-motion support

## Add your LinkedIn link

Open `index.html` and replace:

```text
https://www.linkedin.com/in/YOUR-LINKEDIN-PROFILE
```

with your real LinkedIn profile URL. The GitHub link is already set to `https://github.com/kaniz-codes`.

## Run locally

Open `index.html` directly in a browser, or start a simple local server from the project folder:

```bash
python -m http.server 8000
```

Then open `http://localhost:8000`.

## Deploy with GitHub Pages

1. Upload `index.html`, `style.css` and `script.js` to the root of a GitHub repository.
2. Open **Settings → Pages** in that repository.
3. Under **Build and deployment**, choose **Deploy from a branch**.
4. Choose the `main` branch and `/ (root)` folder, then save.

## Before publishing

The site includes Qur'an excerpts, simplified meanings, selected duas and cited hadith-based practices. Carefully review Arabic text, translations, transliterations and religious explanations with a trusted qualified reviewer before publishing publicly.
